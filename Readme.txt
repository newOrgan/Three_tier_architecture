This is a web movie search application.
The goal of that was searching movies with parameters: by name, by genres or by actor.
Server was builded on .net core Web Api using MSSql database.
Client in its turn was created on vuejs framework using mdbootstrap.

To get start this application at first You should run
sql script "AexDb.sql" in folder "/server".
After that run server on 44321 port.
Then You can start client on your web server
after copying "index.html" and directory "/dist"
to root of your web site.
